# Chatbot - Automação de Mensagens

## 📋 Descrição do Projeto

Chatbot desenvolvido para automatizar atendimentos, responder dúvidas frequentes e organizar o fluxo de contato dos clientes via WhatsApp/site, utilizando lógica de decisão e integrações para oferecer respostas rápidas, claras e personalizadas.

## ⚠️ **AVISO IMPORTANTE - REQUISITOS DO SISTEMA**

> ⚠️ **Este projeto requer Node.js na última versão liberada instalado no seu computador para executar o código.**

### Como instalar Node.js

1. Acesse o site oficial: [https://nodejs.org/](https://nodejs.org/)
2. Baixe a versão **LTS (Long Term Support)** ou a versão mais recente
3. Execute o instalador e siga as instruções
4. Verifique a instalação abrindo o terminal/prompt e digitando:
   ```bash
   node --version
   npm --version
   ```

## ✨ Funcionalidades

- 🤖 Resposta automática de dúvidas frequentes
- 📴 Integração com WhatsApp
- 📄 Gerenciamento de fluxo de atendimento
- 💡 Lógica de decisão inteligente
- 🌟 Personalização de respostas
